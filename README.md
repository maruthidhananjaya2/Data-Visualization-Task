Data Visualization & Storytelling - Power BI Dashboard

#Project Overview
This project demonstrates data visualization and storytelling skills using Power BI to analyze Superstore sales data and derive actionable business insights.

# Objective
Create interactive visualizations that convey a compelling story about sales performance, profitability, and business trends.

# Dataset
- **Source**: Superstore Sales Dataset
- **Content**: Order details, sales, profit, customer segments, product categories, and regional data
- **File**: `Sample - Superstore.csv`

# Visualizations Created

1. **KPI Cards** - Total Sales, Total Profit, Total Quantity, Number of Orders
2. **Sales Trend Over Time** - Line chart showing monthly/yearly sales patterns
3. **Sales by Category** - Bar chart comparing product categories
4. **Profit by Region** - Geographic map showing regional performance
5. **Top 10 Products** - Best-selling products analysis
6. **Profit Margin by Sub-Category** - Detailed profitability breakdown
7. **Sales by Segment** - Customer segment analysis

# Key Insights

- **Technology category** generates the highest revenue
- **West region** shows the strongest profit margins
- **Office Supplies** has the highest order volume
- **Q4 (Oct-Dec)** shows seasonal sales peaks
- **Consumer segment** contributes the majority of sales
